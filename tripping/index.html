Tripping web application

user experience, frontend development
HTML5, CSS3, Javascript (JQuery, JQueryUI), AJAX, Photoshop, Illustrator, Balsamiq

Tripping is a travel site that makes it easy to search, compare and book vacation rentals and short term rentals.

This is a one-month and part-time project, for which I was the sole designer and developer.

Before I came to Tripping, they only had a full version website for desktop. My job was mainly to develop a "mobile solution". Due to the company's time constraint and my availability as a concurrent full-time student at UCBerkeley, a mobile web App was chosen over a native App.

Starting with user research, I had my wireframe done in the first week.

In the following two weeks, I conducted usability testings and worked out the interfaces. In the process we decided to temporarily remove the map feature in order to get it online as soon as possible.

The development was done in the last week, with the help of our CTO and web developer.
